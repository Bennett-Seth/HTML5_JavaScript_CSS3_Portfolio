Focus Events

Events: onfocus

An "onfocus" event occurs when an element enters focus. In other words, when the user clicks into an element.

Enter your name:

For example, when you add your name, a function will changes the background-color.

Events: onblur

An "onblur" event occurs when an element looses focus. In other words, when the user stops interacting with the element.

Enter your name:

When the input field loose focus (when you add text and click away), a function will change your text to the all caps format.
