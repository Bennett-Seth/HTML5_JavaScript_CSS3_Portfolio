7. Fill-Mode for Animations

Property: animation-fill-mode

Value: none (default value which won't apply animation elements before or after execution)

Value: forwards (element retains the style values used in the last keyframe)

Value: backwards (element retains the style values used in the first keyframe)

Value: both (element retains the style values used in both the first and last keyframes)


Value: Forwards. In the example below, the square will retain the position of last keyframe.



Value: Backwards. In the example below, the square will retain the position and color of the first keyframe.
